# SQL Server DBA Portfolio

A high-performance personal portfolio built with Next.js, GSAP, Three.js, and CSS Modules. Designed to showcase SQL Server DBA skills, full stack development experience, and AI-assisted development workflows.


**GitHub:** https://github.com/chawanabhi/Abhishek-cinematic-portfolio.git  

If this project helps you, please consider leaving a ⭐ on GitHub.

## 🚀 Stack

| Layer      | Technology |
| ---------- | ---------- |
| Framework  | Next.js (App Router) |
| Animations | GSAP + Three.js |
| Styling    | CSS Modules |
| Icons      | react-icons |
| Fonts      | Google Fonts via next/font |
